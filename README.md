# SkillSwap – Local Skill Exchange Platform

## Project Purpose

SkillSwap is a website where people can **learn, teach, or trade skills** locally.  
Examples: English lessons, guitar, coding help, yoga, or cooking classes.

Users can browse skills, see details, rate experiences, and connect with skill providers.

---

## Live Link

[View Live Site](https://skillswap-ce11c.web.app/)

## Key Features

- **Sign Up / Login** with Email and Google
- **Protected Pages**: Only logged-in users can see skill details and profile
- **View Skill Listings**: Image, name, price, rating, category
- **Update Profile**: Change name and photo
- **Forgot Password**: Reset password via email
- **Responsive Design**: Works on mobile, tablet, and desktop
- **Animations**: Smooth effects with AOS & Framer Motion

---

## NPM Packages Used

- **react** – Core library for building UI
- **react-router-dom** – Navigation between pages
- **firebase** – Authentication & database
- **react-toastify** – Toast notifications
- **tailwindcss** – Styling
- **daisyui** – UI components
- **swiper** – Slider for homepage hero section
